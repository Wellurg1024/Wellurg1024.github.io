001 - 为什么？

在为什么之前....

Linux是一个开源的操作系统内核，诞生于1991年，由芬兰计算机科学家Linus Torvalds创建。 Linux系统基于类Unix的多用户、多任务操作系统，可以运行在各种硬件平台上， 包括个人电脑、服务器、嵌入式设备等。Linux系统以其稳定性、安全性和自由开放的特点， 受到广泛的欢迎和应用。那么，究竟为什么使用Linux？让我们一起来探讨。

它的特点....

1. 互联网精神

Linux系统的诞生和发展，体现了互联网精神的核心价值：分享、合作、自由。Linus Torvalds在创造Linux系统的初衷并不是商业利益，而是出于对技术的热爱和对自由的追求。他将Linux内核开源，允许任何人查看、修改和分享源代码，这种开放的模式促进了全球开发者的合作，为Linux系统的壮大和发展提供了坚实的基础。

2. 开源精神

Linux系统采用了开源的模式，即软件的源代码对所有人开放。这意味着任何人都可以查看、修改、复制和分享Linux系统的源代码，促进了技术的共享和传播。开源的特点使得Linux系统更加灵活和可定制，用户可以根据自己的需求对系统进行定制和优化，满足不同用户的需求。

3. 自由与安全

Linux系统以其自由和安全的特点成为许多用户的首选。自由意味着用户可以自由选择使用Linux系统，并且可以自由修改和分享系统源代码。这种自由的特点使Linux系统不受商业厂商更新的控制（譬如众所周知并且臭名昭著的Windows更新），用户可以自主决定系统。同时，Linux系统以其稳定性和安全性著称，受到许多企业和政府机构的青睐。

为什么不用Windows？

Linux系统的优势

  1. 开源性：Linux系统是开源的，用户可以自由查看、修改和分享源代码。
  2. 稳定性和安全性：Linux系统以其稳定性和安全性著称，不易受到病毒攻击和系统崩溃。
  3. 灵活性和可定制性：用户可以根据自己的需求对Linux系统进行定制和优化。
  4. 免费和开放：Linux系统可以免费获取和使用，用户可以自由修改和分享系统源代码。
  5. 社区支持和技术更新：Linux系统拥有庞大的开源社区和活跃的开发者团队。

Windows系统的优势

  1. 用户友好性：Windows系统在用户界面设计和易用性方面较为突出。
  2. 广泛的应用支持：Windows系统拥有大量的应用程序和游戏支持。
  3. 商业软件支持：许多商业软件和软件开发工具都是基于Windows平台开发的。
  4. 硬件兼容性：Windows系统对各种硬件设备的兼容性较好。

对于一个服务器，我们当然更看重成本和效率，Linux开源的设计还让他的源代码比Windows好了不少，这让它可以在很多不怎么好的服务器上也可以运行，所以，是的，选择Linux！

同时，当你熟练运用Linux简单可读性高的命令行之后，你会被它快过图形化界面几百倍的工作效率震惊！
