003 - 如何给服务器增加内容

JavaScript、CSS、HTML 是什么？

JavaScript 是一种用于在网页上实现交互和动态效果的编程语言。 它可以让你的网页具有更丰富的功能和用户体验。

CSS（层叠样式表）用于控制网页的样式和布局。 它可以定义字体、颜色、边距、背景等属性，使网页更加美观和易于阅读。

HTML（超文本标记语言）是一种用于创建网页结构和内容的标记语言。 它由一系列的标签组成，每个标签都有不同的含义和作用。

一个网页的基本结构

一个典型的 HTML 网页由以下几个部分组成：

  1. <!DOCTYPE> 声明：它指定了文档类型和版本。
  2. <html> 元素：它是 HTML 文档的根元素。
  3. <head> 元素：它包含了网页的元数据，如标题、样式表和脚本等。
  4. <body> 元素：它包含了网页的可见内容，如文本、图像和其他元素。

下面是一个简单的网页示例： 

<!DOCTYPE html>
        <html>
        <head>
            <title>我的网页</title>
            <link rel="stylesheet" href="styles.css">
            <script src="script.js"></script>
        </head>
        <body>
            <h1>欢迎来到我的网页！</h1>
            <p>这是一个简单的网页示例。</p>
        </body>
        </html>

开始写一个网页吧！

要编写一个简单的前端网页，按照以下步骤进行：

  1. 创建一个新的 HTML 文件，例如 index.html。
  2. 使用文本编辑器打开该文件，并将上述示例代码复制到文件中。
  3. 根据需要修改 <title> 元素中的标题和 <h1> 元素中的欢迎消息。
  4. 在 <body> 元素中编写你的网页内容，可以使用其他 HTML 标签来创建段落、链接、图像等元素。
  5. 如果需要，可以创建一个单独的 CSS 文件（例如 styles.css）来定义网页的样式。
  6. 如果需要添加交互和动态效果，可以创建一个单独的 JavaScript 文件（例如 script.js）来编写相关代码。
  7. 在浏览器中打开该 HTML 文件，查看你的网页效果。

如何上传到你的 Apache2 服务器上

要将你的网页上传到 Apache2 服务器上，可以按照以下步骤进行：

  1. 确保你的 Apache2 服务器已正确安装和配置。 你可以使用命令 sudo service apache2 status 来检查 Apache2 服务器的状态。
  2. 将你的 HTML 文件、CSS 文件和 JavaScript 文件复制到 Apache2 服务器的网站根目录。 默认情况下，该目录在 Linux 上为 /var/www/html/。
  3. 使用终端或 SSH 连接到你的服务器，并导航到网站根目录。
  4. 使用以下命令将文件复制到网站根目录：
sudo cp index.html styles.css script.js /var/www/html/
  1. 确保复制的文件具有正确的权限，以便 Apache2 可以读取和提供它们：
sudo chown -R www-data:www-data /var/www/html/
        sudo chmod -R 755 /var/www/html/
  1. 现在，你的网页已经上传到 Apache2 服务器上。 你可以通过输入服务器的 IP 地址或域名来访问你的网页。 在浏览器中输入 http://服务器IP地址:80/，就可以查看你的网页了。

请注意，这里提供的步骤和命令是基于默认配置的 Apache2 服务器。
如果你的服务器有特定的配置或使用其他 Web 服务器，可能需要进行适当的调整。
另请注意：这只是在你的本地局域网架设的服务器，在以后，我们会让整个互联网的用户都能访问他。
