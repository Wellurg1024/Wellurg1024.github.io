002 - 如何开设一个服务器

安装Apache Web服务器

  1. 打开终端并以管理员身份登录到Debian系统。
  2. 运行以下命令来更新软件包列表：
sudo apt update
  1. 安装Apache Web服务器：
sudo apt install apache2
  1. 安装过程中，您可能需要提供管理员密码以完成安装。
  2. 安装完成后，Apache将自动启动并开始监听默认的HTTP端口( 80 )

配置防火墙

  1. 如果您的Debian系统上启用了防火墙(例如ufw),您需要允许HTTP流量通过。运行以下命令来允许入站HTTP连接：
sudo ufw allow 'Apache'

如果您使用的是其他防火墙工具，请根据其文档进行相应的配置。

启动和测试Web服务器

  1. 启动Apache Web服务器：
sudo service apache2 start
sudo service apache2 restart
